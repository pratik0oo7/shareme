Sharik - ทางออกสำหรับการแชร์ไฟล์ภายในเครือข่าย ที่โอเพนซอร์ส และรองรับหลากหลายแพลตฟอร์ม


แอพนี้จะช่วยให้คุณส่งภาพ แอพ ไฟล์ ข้อความ หรืออะไรก็ตาม ได้อย่างง่ายดายไปยังอุปกรณ์ต่าง ๆ ที่เชื่อมต่อในเครือข่ายอินเทอร์เน็ตเดียวกัน สะดวกสบายและรวดเร็วทันใจ


หลังจากการเลือกภาษาและผ่านคำแนะนำการใช้งาน คุณจะพบปุ่มแชร์และไฟล์ล่าสุดที่คุณเคยแชร์อยู่บนหน้าจอหลัก จากนั้นคุณก็สามารถเลือกแชร์ภาพ ไฟล์ แอพที่ติดตั้งอยู่ในเครื่อง หรือข้อความง่าย ๆ ได้ทันที หลังจากที่คุณเลือกไฟล์ คุณจะเห็นหน้าจอแชร์ ซึ่งจะมีข้อมูลเกี่ยวกับการเชื่อมต่อของคุณ (Wi-Fi หรือ mobile hotspot) และลิงค์ที่คุณจะต้องใช้ในเครื่องที่จะรับไฟล์ หลังจากใส่ URL ไปในเบราว์เซอร์และกด enter บนเครื่องผู้รับ ไฟล์จะถูกดาวน์โหลดทันที โปรดยืนยันให้แน่ใจว่าทั้งสองเครื่องเชื่อมต่ออยู่บนเครือข่ายเดียวกัน


หากคุณติดตั้ง Sharik ไว้บนทั้งสองเครื่องแล้ว ในการรับไฟล์จากเครื่องอื่น คุณสามารถกดปุ่ม "รับข้อมูล" บริเวณด้านล่างของหน้าจอบนเครื่องที่จะใช้รับไฟล์ได้ หลังจากที่คลิกแล้ว หน้าจอของผู้รับจะปรากฏขึ้น ถ้าแอพ Sharik พบเครื่องอื่นที่กำลังจะแชร์ข้อมูลอยู่ในเครือข่ายของคุณ แอพจะแสดงอุปกรณ์นั้นขึ้น จากนั้นคุณก็แต่คลิกเลือกอุปกรณ์ หน้าต่างเบราว์เซอร์จะปรากฏ และเริ่มการดาวน์โหลด


Sharik นั้นเป็นแอพที่ใช้งานได้ฟรี และไม่มีโฆษณา มากไปกว่านั้น โค้ดของแอพยังมีการเผยแพร่บน GitHub อีกด้วย ทุกคนจึงสามารถมีส่วนร่วมในการพัฒนาได้ หากคุณชอบ Sharik โปรดกด star ใน repository ที่อยู่บน GitHub และหากเป็นไปได้ มาร่วมเป็นส่วนหนึ่งในการพัฒนา Sharik กับเรา ตัวอย่างเช่น เพิ่มคำแปลภาษา หรือช่วยพัฒนาโค้ดของแอพ


แอพ Sharik เขียนด้วย Dart, Flutter, Hover.build และ Golang และเผยแพร่อยู่ภายใต้สัญญาแบบเปิด MIT
